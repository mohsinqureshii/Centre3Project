# Centre3 Admin Setup Guide

1) Login as SUPER_ADMIN or ADMIN
2) Settings → Locations: create Country/Region/City/Site
3) Settings → Zones: create zones for each site
4) Settings → Rooms: create rooms under zones
5) Settings → Activities: create categories and activities
6) Settings → Processes: define approval processes and stages
7) Settings → Users: create users and assign roles
8) Validate: create a request, submit, approve, and issue credential
